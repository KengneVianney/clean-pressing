# 🧺 CleanPressing — Application de Gestion de Pressing

Projet ICT202 — Développement Mobile  
Groupe 9x — 8 membres (4 binômes)

---

## 📌 Présentation

Application web de gestion de pressing permettant :
- gestion des commandes clients
- suivi des statuts
- calcul automatique des prix
- dashboard de statistiques

---

## 🛠️ Stack technique

- React + TypeScript
- React Router
- IndexedDB
- Yup (validation)
- Recharts (dashboard)

---

## 👥 Répartition des tâches

### Binôme 1 — UI & Navigation
- Layout (sidebar + navbar)
- Routing
- Pages de base

### Binôme 2 — Base de données
- IndexedDB
- CRUD commandes
- Hook useOrders

### Binôme 3 — Formulaires
- Formulaire commande
- Validation Yup
- Calcul prix

### Binôme 4 — Dashboard
- Statistiques
- Graphiques
- UI finale

---
